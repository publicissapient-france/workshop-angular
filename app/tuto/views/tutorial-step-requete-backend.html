AngularJS offre la possibilité de requêter un backend de différentes manières, parmi elles, on trouve le service $http.

Cette étape a pour objectif de supprimer les logs déclarées en dur dans le contrôleur et de requêter un backend pour les récupérer.

Pour ce faire :

  • Dans le fichier workshop-angular/app/js/app.js, s'injecter le service $http au sein du contrôleur
  • Requêter les logs à l'URL /logs (prendre soin d'avoir démarrer son serveur Node (node server à la racine du projet avec un npm install au préalable pour installer les dépendances)
  • Stocker les logs récupérées au sein du $scope
  • Invoquer la fonction filterLogs précédemment implémentée au cas où un filtre serait décoché
